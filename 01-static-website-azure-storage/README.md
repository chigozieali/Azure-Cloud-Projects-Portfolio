# Project 1: Static Website Hosted on Azure Blob Storage

**Project Goal**: Deploy a personal portfolio website using Azure Storage static website hosting (no web server needed).

## Technologies Used
- Azure Storage Account (General Purpose v2)
- Azure Blob Storage + $web container
- Static Website Hosting feature
- HTML + CSS

## What I Built
- Enabled static website hosting on a storage account
- Created and uploaded index.html and 404.html
- Made the site publicly accessible via Azure's primary endpoint

## Live Website
[Click here to view live site]https://chigoziestatic2026.z6.web.core.windows.net/)


## Key Learnings
- How static website hosting works in Azure Storage
- Difference between regular containers and the $web container
- Public endpoints and anonymous read access
- Why this is a cheap & fast way to host static content
